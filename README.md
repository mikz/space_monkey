# SpaceMonkey

Ruby Client for Space Monkey API.

Experimental. Space Monkey API is not public yet and it can change any time.

## Installation

Add this line to your application's Gemfile:

    gem 'space_monkey'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install space_monkey
